# EVE Front End
## Tech Stack
React + JavaScript
## Run Instruction
### `npm install`
This downloads required node_modules with React
### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
### `npm run build` (For Production)
